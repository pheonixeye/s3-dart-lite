# s3-dart-lite

A lightweight S3 client for Dart, ported from the excellent TypeScript [s3-lite-client](https://github.com/bradenmacdonald/s3-lite-client).

!!! warning "Alpha Stage"
    This project is currently in the alpha stage and is NOT ready for production use. APIs may change.

## Features

- **Lightweight**: Minimal dependencies.
- **S3 Compatible**: Works with AWS S3, MinIO, DigitalOcean Spaces, etc.
- **Dart Native**: Built with Dart idioms (Streams).
